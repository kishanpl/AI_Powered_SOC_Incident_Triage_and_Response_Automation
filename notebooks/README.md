# Model Training (Google Colab)

The ANN/CNN model training is done in Google Colab due to GPU requirements.

## Steps to train:
1. Upload CICIDS2017 CSV files to Google Drive
2. Open the training notebook in Colab
3. Train model and download the saved `.pkl` or `.h5` file
4. Place the model file in the `models/` directory
5. The dashboard will automatically load it for predictions

## Notebooks:
- `SOC_Model_Training.ipynb` — Main training notebook (run in Colab)
